Neural Networks

Explanation of Model

A Neural Network is a type of machine learning - where a computer learns to perform a task by studying training examples.

  1. Split our data into a training set and a testing set
  2. Scaled our data
  3. Label encoded and one hot encoded the class name for each animal
  4. Complied and fit the model
  5. Calculated loss and accuracy of the model

We weren't sure whether or not to scale our data since it was already all numbers. To be thorough, we tested both the Standard and MinMax scaling as well as unscaled data.

The results for all 6 combinations of models turned out very similar - with a high degree of accuracy. The most accurate model seemed to be the one using unscaled data. All of the models had a hard time identifying turtles. The MinMax model was unable to classify the spider correctly.
