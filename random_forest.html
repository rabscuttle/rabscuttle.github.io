Random Forest


Explanation of Model

The Random Forest Classifier is a supervised machine learning model. It works by combining the results of a large number of individual decision trees. After running the unknown set of data through multiple decision trees, the classification is predicted by taking the majority answer from the individual decision trees.

For our model, we decided to use an estimator of 50. Increasing the estimator beyond this point didn't have a noticable difference on the results.
