Unsupervised K-Means

Explanation of Model

K-means is an unsupervised learning algorithm and is sometimes referred to as Lloyd’s algorithm or "naive k-means". This machine learning model is typically used with unlabeled or unclustered data. The purpose of the K-means is to identify groups or “clusters” in the data. The number of clusters the model will identify is represented by K and requires the number of clusters to be specified. The algorithm assigns each data point to one of the K groups based on the features in the data set. Data points are clustered based on feature similarity. Typically, K-means is used with unlabeled data, we wanted to see if a k-means model could successfully place animals in the appropriate classes with the features provided. The steps to using this model were:

  1. Select number of clusters (K = 7)
  2. Fit the model to the data
  3. Predict the clusters

Principle Component Analysis (PCA)

Plotting clusters became a challenge, as we had more than 2 features to visualize on a scatter plot. Therefore, we ran a principal component analysis (PCA). A PCA reduces the number of variables of a data set, while preserving as much information as possible. It does this by combining our input variables in a specific way to create new variables. Because of the way the new variables are combined, we can drop the least important variables while still retaining the most valuable parts of all of the variables. We used the PCA to reduce our 16 features and then plotted the following visualizations:

  1. 1-D Visualization: Displays our clusters on one principal component
  2. 2-D Visualizations: Displays our clusters on two principal components
  3. 3-D Visualizations: Displays our clusters on three principal components
