Support Vector Machine


Explanation of Model

Support Vector Machine is a supervised machine learning model. It finds the hyperplane with the maximum margin between the data classes in order to predict the classification of new data points.

Because of the variety of parameters used as part of the SVM model, we used the Grid Search module included in Scikit-learn to test different parameter values. None of the parameters we tested returned a better score than the defaults of the model.
